# Звуковые файлы для игры

Эта директория содержит звуковые файлы, которые используются в игре.

## Необходимые файлы:

1. **jump.mp3** - Звук прыжка утки
2. **hit.mp3** - Звук столкновения
3. **score.mp3** - Звук набора очков

## Как получить звуки:

### Вариант 1: Сгенерировать программно

1. Перейдите в `src/assets/sounds/`
2. Запустите скрипт генерации:
   ```bash
   node generate-sounds.js
   ```
3. Скрипт создаст WAV файлы
4. Конвертируйте WAV в MP3 (см. инструкции в `src/assets/sounds/README.md`)
5. Скопируйте MP3 файлы в эту директорию (`public/sounds/`)

### Вариант 2: Использовать готовые звуки

Скачайте звуки с бесплатных ресурсов (см. `src/assets/sounds/README.md`) и поместите их в эту директорию с именами:
- `jump.mp3`
- `hit.mp3`
- `score.mp3`

## Примечание:

Звуки загружаются автоматически при запуске приложения через `SoundManager` (см. `src/main.tsx`).
