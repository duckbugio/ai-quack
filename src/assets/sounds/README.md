# Звуковые эффекты для игры "Утка"

Эта директория содержит звуковые файлы для игры.

## Необходимые звуки:

1. **jump.mp3** - Звук прыжка утки (короткий, высокий звук)
2. **hit.mp3** - Звук столкновения (резкий, низкий звук)
3. **score.mp3** - Звук набора очков (приятный, короткий звук)
4. **background.mp3** - Фоновая музыка (опционально, спокойная мелодия)

## Где найти звуки:

### Бесплатные ресурсы:
- **Freesound.org** - https://freesound.org/ (требуется регистрация)
- **OpenGameArt.org** - https://opengameart.org/
- **Zapsplat** - https://www.zapsplat.com/ (требуется регистрация)
- **Mixkit** - https://mixkit.co/free-sound-effects/

### Рекомендуемые поисковые запросы:
- Прыжок: "jump", "flap", "wing flap", "whoosh"
- Столкновение: "hit", "crash", "collision", "impact"
- Очки: "coin", "pickup", "success", "ding"
- Фоновая музыка: "game music", "background loop", "ambient"

## Формат файлов:

- Рекомендуемый формат: **MP3** (хорошая компрессия)
- Альтернатива: **OGG** (лучшая компрессия, но не все браузеры поддерживают)
- Размер файлов: стараться держать каждый файл < 100KB для быстрой загрузки

## Генерация звуков программно:

Если у вас нет готовых звуковых файлов, можно использовать скрипт `generate-sounds.js`
для программной генерации базовых звуковых эффектов.

### Использование скрипта:

1. **Запуск генерации:**
   ```bash
   node src/assets/sounds/generate-sounds.js
   ```

2. **Результат:**
   Скрипт создаст три WAV файла:
   - `jump.wav` - звук прыжка
   - `hit.wav` - звук столкновения
   - `score.wav` - звук набора очков

3. **Конвертация в MP3 (рекомендуется):**
   
   Для лучшей совместимости и меньшего размера файлов рекомендуется конвертировать WAV в MP3:
   
   **С помощью ffmpeg:**
   ```bash
   ffmpeg -i jump.wav jump.mp3
   ffmpeg -i hit.wav hit.mp3
   ffmpeg -i score.wav score.mp3
   ```
   
   **Онлайн-конвертеры:**
   - https://cloudconvert.com/wav-to-mp3
   - https://convertio.co/wav-mp3/
   - https://www.freeconvert.com/wav-to-mp3

4. **Размеры файлов:**
   - Сгенерированные WAV файлы обычно занимают ~10-20 KB каждый
   - После конвертации в MP3 размер уменьшится до ~5-10 KB

## Использование в игре:

Звуки будут загружены через `SoundManager` (см. этап 9.2 в todo.md).

## Примечание:

Сгенерированные звуки являются базовыми и подходят для прототипирования.
Для финальной версии игры рекомендуется использовать профессиональные звуки
из бесплатных ресурсов, перечисленных выше.
