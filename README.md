# Утка - Flappy Bird Clone

Веб-игра в стиле Flappy Bird, разработанная на React и TypeScript. Цель игры - управлять уткой, избегая препятствий и набирая очки.

## Описание

"Утка" - это современная веб-версия классической игры Flappy Bird. Игрок управляет уткой, которая должна пролетать между препятствиями, набирая очки за каждое успешно пройденное препятствие. Игра включает прогрессивное увеличение сложности, систему сохранения лучшего результата, звуковые эффекты и адаптивный дизайн для различных устройств.

## Технологии

- **React 18+** - UI библиотека для создания компонентов
- **TypeScript 5+** - Типизированный JavaScript для надежности кода
- **Vite** - Современный сборщик для быстрой разработки
- **Canvas API** - Рендеринг игровой графики и анимаций
- **CSS Modules** - Стилизация компонентов
- **Vitest** - Фреймворк для тестирования

## Установка

1. Клонировать репозиторий:
   ```bash
   git clone <repository-url>
   cd duck-game
   ```

2. Установить зависимости:
   ```bash
   npm install
   ```

## Запуск

### Режим разработки

Запустить dev-сервер с hot-reload:
```bash
npm run dev
```

Игра будет доступна по адресу `http://localhost:5173` (или другому порту, указанному Vite).

### Production сборка

Создать оптимизированную сборку для production:
```bash
npm run build
```

Собранные файлы будут находиться в директории `dist/`.

### Предпросмотр production сборки

Запустить локальный сервер для предпросмотра production сборки:
```bash
npm run preview
```

## Управление

- **Пробел** или **Стрелка вверх** - прыжок утки
- **Клик мыши** или **Тап** - прыжок утки
- **Escape** - пауза (во время игры)

## Игровые механики

- **Гравитация** - утка постоянно падает вниз под действием гравитации
- **Прыжок** - нажатие клавиши или клик поднимает утку вверх
- **Препятствия** - случайно генерируемые препятствия с проходами
- **Подсчет очков** - очки начисляются за каждое пройденное препятствие
- **Прогрессивная сложность** - скорость препятствий увеличивается с ростом счета
- **Сохранение рекорда** - лучший результат сохраняется в localStorage

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── Game/           # Основной игровой компонент
│   └── UI/             # UI компоненты (меню, кнопки)
├── contexts/           # React контексты для состояния
├── game/              # Игровая логика
│   ├── entities/      # Игровые сущности (Утка, Препятствия)
│   ├── systems/       # Игровые системы (физика, коллизии)
│   └── utils/         # Утилиты (константы, хелперы)
├── assets/            # Ресурсы (изображения, звуки)
├── hooks/             # Кастомные React хуки
├── types/             # TypeScript типы и интерфейсы
└── test/              # Настройки тестового окружения
```
